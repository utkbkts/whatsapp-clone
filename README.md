## ⚙️ Projeyi çalıştır
1. **First, clone the repository:**
```bash
git clone https://github.com/utkbkts/whatsapp-clone.git
```
2.**Yönlendirmeler ve komutlar:**
 ```bash
cd frontend
npm install
```
3.**Projeyi çalıştır:**
 ```bash
npm run dev
```
4.**Local'de derle:**
```bash
npm run build
```
